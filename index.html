Spotify Usability Test

Test 1: Create an Account

Time: 15 seconds

Found the "sign up" button immediately, which brings the user to a menu to create an account

Test 2: Download the Desktop App

Time: 5 seconds

The download button is in a very similar spot compared to the sign up button, so the user found it very quickly

Test 3: Find Payments and Cancellations

Time: 1 minute and 50 seconds

The terms and conditions is relatively hard to find at the bottom of the page, and then the pricing is then even harder to find within the terms and conditions.

Test 4: Find Their Instagram

Time: 10 seconds

The link was right at the bottom of the page with the Instagram logo. However, this would be harder for someone if they did not know what the Instagram logo looked like.

Test 5: Find the API to Pull and Album

Time: 45 seconds

There were a few misclicks along the way but the page is set up so that even if you go to the wrong page, the footer is still the same, so the user found their way to the API page.
